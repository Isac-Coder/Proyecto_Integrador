Aquí se explica como crear, guardar y montar una imagen de un contenedor en docker

1. Guarda el contenedor como una imagen:

``` bash
    docker commit nombre-contenedor nombre-nueva-imagen
```
2. Exporta la imagen como un archivo .tar:

``` bash
    docker save -o mi_imagen.tar nombre-nueva-imagen
```
Nota: La imagen se guardara en la carpeta en la que estes ubicado

3. Listo para pasarla y montarla en otra maquina usando:

``` bash
    docker load -i mi_imagen.tar
```
